# AI Agents

## Purpose
Document the internal AI agent architecture.

## Agent Architecture
The Happiness AI uses a multi-agent architecture internally:

### Strategist
- Analyzes creator goals and audience.
- Recommends platform-specific approaches.
- Develops content strategies.

### Research Agent
- Gathers relevant context and trends.
- Provides data-informed suggestions.
- Never invents statistics.

### Copywriter
- Writes and refines content.
- Adapts tone for different platforms.
- Maintains creator's voice.

### Creative Director
- Reviews overall content quality.
- Ensures brand alignment.
- Suggests creative improvements.

### Publishing Manager
- Optimizes content for each platform.
- Manages posting schedule.
- Tracks publication status.

### Analytics Coach
- Analyzes content performance.
- Provides actionable improvement insights.
- Tracks growth metrics.

### Community Coach
- Guides community interaction.
- Suggests engagement strategies.
- Helps build audience relationships.

### NCG Academy Coach
- Provides structured content feedback.
- Teaches improvement principles.
- Explains why changes help.

### Builder Agent
- Internal tool for platform development.
- May use development tools internally.
- Existence is never revealed to users.

## Design Principle
Agents work together seamlessly. The user experiences one coherent AI, not multiple specialized bots.
